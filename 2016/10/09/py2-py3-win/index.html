python3作为主要版本的话，只需将python2安装目录下的python.exe 更名为python2.exe即可。

使用两个版本的pip

或者更简单的

但使用pip2 --version时可能会遇到问题：

Fatal error in launcher: Unable to create process using ‘“‘

原因是python2的pip收到了损坏。
修复的方法很简单：升级pip就好了。

python2 -m pip install -upgrade pip
